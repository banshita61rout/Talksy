# 🚀 About The Project

Talksy is a full-stack real-time video conferencing web application that allows users to create and join meeting rooms instantly.

Built using **WebRTC** and **Socket.IO**, Talksy enables peer-to-peer video/audio communication with real-time signaling, live chat, authentication, and meeting history.

The platform is designed to provide a smooth and modern meeting experience directly inside the browser — without requiring downloads or installations.

# ⚡ Quick Start Guide

### ✅ Works on Windows, Mac & Linux

### ⏱️ Setup Time: 2–5 Minutes

---

# 🔧 1. Clone The Repository

```bash
git clone https://github.com/banshita61rout/Talksy.git
```

# 📁 2. Navigate To Project Folder

```
cd Talksy

```

# 📦 3. Setup Backend

```
cd Backend
npm install
```

### Create a .env file:

```
PORT=8000
MONGO_URI=your_mongodb_connection_string
```

#### Run backend:

```
npm run dev

```

# 💻 4. Setup Frontend

```
cd ../frontend
npm install
npm run dev

```

### Frontend runs on:

```
http://localhost:5173

```

## 🛠️ Tech Stack

### 🎨 Frontend

- React.js
- React Router
- CSS Modules
- Vite

### ⚙️ Backend

- Node.js
- Express.js

### 🔄 Real-Time Communication

- Socket.IO
- WebRTC

### 🗄️ Database

- MongoDB Atlas
- Mongoose

### 🔐 Authentication

- bcrypt password hashing

### ☁️ Deployment

- Netlify (Frontend)
- Render (Backend)

## 🔗 Application Flow

User → React Frontend → Socket.IO Signaling → WebRTC Connection → Peer-to-Peer Video Call

## 🎯 Core Features

## 🎥 Real-Time Video Calling

- Create meeting rooms instantly
- Join using shared room links
- Seamless peer-to-peer communication

## 👥 Multi-Participant Meetings

- Multiple users can join the same room
- Dynamic participant handling

## 💬 Live Chat System

- Real-time messaging during meetings
- Instant communication using Socket.IO

## 🖥️ Screen Sharing

- Share your screen during meetings
- Useful for presentations & collaboration

## 🎙️ Media Controls

- Toggle microphone
- Toggle camera
- Manage audio/video streams

## 🔐 Authentication System

- User registration & login
- Secure password hashing using bcrypt
- 📜 Meeting History
- Previous meetings stored in database
- Easy tracking of joined meetings
